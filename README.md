# DrinksShop-Project
This a desktop app where we present the available drinks in the shop. There's a login page where we allow the admin to connect ana to generate the app
if the user doesn't have an account he can go to the sign up page to create a new acount.
The first dashboard presents the trending drinks. The second dashboard named "manage" is the page that contains all the drinks that exists in the store
The admin can surely manipulate those drinks by adding, deleting or  modifying a product he can also generate the bill and printing it.
